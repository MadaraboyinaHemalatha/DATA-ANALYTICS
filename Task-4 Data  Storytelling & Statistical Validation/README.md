# Data Analytics Internship Project

## Project Overview
This project focuses on analyzing sales data to identify business insights using data analytics techniques. The project includes data cleaning, exploratory data analysis (EDA), dashboard creation, and data storytelling.

## Objectives
- Clean and prepare the dataset.
- Perform exploratory data analysis.
- Build an interactive dashboard.
- Identify key business insights.
- Provide business recommendations.

## Tools Used
- Microsoft Excel
- Google Sheets
- Power BI
- GitHub
- Microsoft PowerPoint

## Project Tasks
### Task 1 – Data Immersion & Wrangling
- Data cleaning
- Missing value handling
- Duplicate removal

### Task 2 – Exploratory Data Analysis
- Sales trend analysis
- Category analysis
- Customer analysis

### Task 3 – Interactive Dashboard
- KPI dashboard
- Charts and graphs
- Business insights

### Task 4 – Data Storytelling & Statistical Validation
- Business story
- Recommendations
- Statistical validation

## Key Insights
- Total Sales: ₹139.24 Million
- Grocery category generated the highest sales.
- Male and female sales were nearly equal.
- Monthly sales showed seasonal trends.

## Business Recommendations
- Increase inventory for high-selling products.
- Improve marketing for low-performing categories.
- Run seasonal promotional campaigns.
- Focus on customer retention.
